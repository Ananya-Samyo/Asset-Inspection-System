## 📖 About The Project

**Asset Inspection System** เป็นเว็บแอปพลิเคชันที่ช่วยให้ผู้ใช้งานสามารถ **ตรวจสอบ จัดระเบียบ และติดตามข้อมูลของสินทรัพย์ (assets)** ภายในองค์กรอย่างเป็นระบบและมีประสิทธิภาพ  
ระบบนี้สร้างด้วย **Vue 3 + Vite** เพื่อให้มี UI ที่ทันสมัยและตอบสนองได้ดีสำหรับผู้ใช้งาน

โปรเจคนี้เหมาะสำหรับการ

- จัดเก็บรายการสินทรัพย์
- บันทึกสถานะการตรวจสอบ
- วิเคราะห์หรือแสดงผลการตรวจสอบ
- เตรียมรายงานสำหรับผู้ดูแลระบบ

---

## ✨ Key Features

🔎 **Asset Listing**  
ดูรายการสินทรัพย์ทั้งหมด พร้อมรายละเอียดแต่ละรายการ

📝 **Inspection Management**  
บันทึกและจัดการสถานะการตรวจสอบสินทรัพย์

📊 **Data Overview**  
แสดงภาพรวมสถานะสินทรัพย์ (เช่น inspected / pending / issues)

📁 **Responsive UI**  
รองรับการใช้งานบนอุปกรณ์ต่าง ๆ
